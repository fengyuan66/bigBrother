Reading guide

Agent: should judge the current evidence and decide if it is sufficient.

Planner: should request more resources only when the current evidence is not enough.

Response: should turn a response-worthy plan into the final spoken line, then attach an ElevenLabs voice clip if configured.

Resources: webcam, screenshare, and browser export should be readable without opening raw files.
